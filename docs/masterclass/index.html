üìö What You'll Learn

This masterclass teaches you how to build production-ready AI agents using Python and LangGraph. You'll progress from a simple chatbot to a sophisticated agent with tool usage, persistent memory, and human oversight capabilities.

Perfect for developers who want hands-on experience building AI systems that can reason, use tools, remember context, and involve humans in decision-making loops.

üìì Jupyter Notebooks

Work through these interactive notebooks in sequence:

  • 01 - Introduction

    Build a basic chatbot using LangGraph and learn the fundamentals of graph-based conversation flows.

  • 02 - Tools

    Enhance your chatbot with tool-calling capabilities, enabling it to search the web and perform calculations.

  • 03 - Memory

    Implement persistent memory so your agent can remember and reference past conversations.

  • 04 - Human-in-the-Loop

    Add human oversight with interruption points where users can approve, reject, or modify agent actions.

  • 05 - Research Agent

    A complete implementation applying all concepts: a search agent for business idea validation.

üîë Key Concepts

  • Graph-based Architecture: Build agents as state machines with nodes and edges
  • Tool Integration: Enable agents to call external APIs and functions
  • State Management: Maintain conversation context across interactions
  • Checkpointing: Persist agent state for resumable conversations
  • Human Oversight: Implement approval workflows for critical actions
